# 🎉 Что нового: Три места размещения плагинов!

## ✨ Что изменилось

Теперь плагины могут размещаться в **трех разных местах**:

### 1️⃣ **Sidebar** (Основное меню)
Для навигационных разделов
- Чаты
- Переменные
- История

### 2️⃣ **Footer** (Внизу sidebar)
Для вспомогательных функций
- 📚 Документация (новое!)
- 💳 Биллинг

### 3️⃣ **Dashboard Widgets** (Карточки на главной)
Для полноценных приложений
- 🎨 Flow Builder
- 🤖 Мои боты
- 🏪 Магазин
- 🕒 История

## 🎯 Как работает

### Плагин решает, где размещаться:

```python
class MyPlugin(Plugin):
    # В основном меню
    sidebar_items = [{
        "label": "Мой раздел",
        "icon": "bi-star",
        "url": "/frontend/my/"
    }]
    
    # Внизу sidebar
    footer_items = [{
        "label": "Настройки",
        "icon": "bi-gear",
        "url": "/frontend/settings/"
    }]
    
    # Карточка на главной
    dashboard_widgets = [{
        "title": "Мое приложение",
        "description": "Описание",
        "icon": "bi-app",
        "url": "/frontend/my-app/"
    }]
```

**Можно комбинировать!** Плагин может быть одновременно:
- В sidebar
- В footer
- И виджетом на dashboard

## ✅ Проверь

В консоли браузера (F12):

```javascript
console.log({
  plugins: window.__PLUGINS__?.length,
  loaded: app.pluginManager?.getLoadedNames()?.length,
  builder: !!app.builder,
  bots: !!app.bots,
  store: !!app.store
});
```

**Должно быть:**
```javascript
{
  plugins: 10,   // 10 плагинов
  loaded: 3,     // 3 JS модуля
  builder: true,
  bots: true,
  store: true
}
```
