# Инструкции Sync

Sync — рабочий чат команды с пространствами, каналами, личными диалогами, тредами, файлами, упоминаниями, закрепами, настройками уведомлений и звонками.

Начинайте с основной длинной инструкции: [Sync: полная инструкция по сервису](sync-complete-guide/). Она проходит весь базовый маршрут пользователя: выбор пространства, настройка пространства, создание канала, отправка сообщений, упоминание коллеги, вложение файла, меню сообщения, закреп, тред, настройки канала и профиль участника.

Остальные инструкции ниже — короткие точечные маршруты. Они полезны, когда нужно быстро проверить отдельный экран или конкретное действие.
