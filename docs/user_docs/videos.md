# Видео-гайды

## Быстрый старт

### Регистрация и настройка (3 мин)
- Создание аккаунта
- Настройка компании
- Подключение API ключей

### Первый бот за 5 минут (5 мин)
- Установка из Store
- Подключение Telegram
- Тестирование

## Продвинутые возможности

### Создание бота с нуля (10 мин)
- Выбор модели ИИ
- Написание промпта
- Настройка инструментов

### Подключение платформ (15 мин)
- Telegram интеграция
- WhatsApp Business
- Web-виджеты

### Визуальный конструктор (12 мин)
- Создание сложных сценариев
- Работа с блоками
- Отладка и тестирование

## Специализированные гайды

### Интеграция с CRM (8 мин)
- Настройка AmoCRM
- Синхронизация данных
- Автоматизация процессов

### Оптимизация расходов (6 мин)
- Мониторинг биллинга
- Настройка лимитов
- Эффективные промпты
