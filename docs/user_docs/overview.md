# Обзор платформы Agent Lab

Agent Lab - это современная no-code платформа для создания и управления ИИ-агентами. Платформа позволяет бизнес-пользователям автоматизировать рутинные задачи и создавать интеллектуальных помощников без навыков программирования.

## 🏗 Архитектура платформы

### Что такое Agent Lab

Agent Lab построен на принципах **Database-First** архитектуры:

- **Все конфигурации** агентов, потоков и настроек хранятся в базе данных
- **Единообразие** - агенты из кода и созданные через UI работают идентично
- **Масштабируемость** - легко добавлять новых агентов и интеграции
- **Безопасность** - изоляция данных по компаниям

## 🎯 Возможности платформы

### Для бизнес-пользователей

#### 🤖 Создание ботов
- **Магазин готовых ботов** - установите проверенные решения одним кликом
- **Визуальный конструктор** - drag-and-drop интерфейс для создания логики
- **Готовые сценарии** - шаблоны для типовых задач (поддержка, продажи, FAQ)

#### 🌐 Многоплатформенность
- **Telegram** - чат-боты и группы
- **WhatsApp Business** - официальный API для бизнеса
- **Web-виджеты** - встроенные чаты на сайт
- **API** - интеграция с существующими системами
- **AmoCRM** - автоматизация процессов продаж

#### 📊 Аналитика и мониторинг
- **История диалогов** - все взаимодействия с пользователями
- **Статистика использования** - расходы на ИИ, нагрузка
- **Биллинг** - прозрачное ценообразование pay-as-you-go

### Для разработчиков

#### 🛠 Технические возможности
- **LangGraph-native** - использование современных инструментов
- **Асинхронная архитектура** - высокая производительность
- **Repository Pattern** - чистая архитектура кода
- **Type Safety** - строгая типизация

#### 🔧 Расширяемость
- **Custom Tools** - создание собственных инструментов
- **Custom Agents** - разработка специализированных агентов
- **Middleware** - интеграция с внешними системами
- **Webhooks** - автоматизация бизнес-процессов

## 🚀 Быстрый старт для новых пользователей

### 1. Регистрация
- Вход через **Yandex** или **Google**
- Автоматическая настройка профиля

### 2. Создание компании
- **Название компании** - отображаемое имя
- **Slug** - уникальный идентификатор (mycompany)
- **Поддомен** - `mycompany.agents-lab.ru`

### 3. Выбор решения
- **Магазин** - готовые боты для быстрого старта
- **Создание с нуля** - для специфических задач
- **Builder** - для сложной логики

### 4. Настройка платформ
- **Telegram** - получить токен у @BotFather
- **WhatsApp** - настроить Business API
- **Web** - встроить виджет на сайт

## 💰 Модель оплаты

### Pay-as-you-go
- **Только за использование ИИ** - оплата за токены LLM
- **Прозрачное ценообразование** - стоимость каждого запроса
- **Гибкие лимиты** - от бесплатного тарифа до enterprise

### Тарифные планы
- **Free** - 1000 запросов/месяц
- **Basic** - 990₽/месяц + оплата за ИИ
- **Premium** - 2990₽/месяц + оплата за ИИ
- **Enterprise** - индивидуальные условия
