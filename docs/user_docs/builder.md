## Что такое Builder

Builder - это drag-and-drop конструктор для создания сложных сценариев взаимодействия с ИИ без программирования.

## Возможности

### Визуальное программирование
- Перетаскивайте блоки для создания логики
- Соединяйте компоненты стрелками
- Настраивайте параметры в интерфейсе

### Доступные блоки

#### Агенты
- OpenAI GPT-4
- YandexGPT
- Google Gemini

#### Инструменты
- Поиск в интернете
- Работа с базами данных
- Отправка email
- API вызовы

#### Логика
- Условия (if/else)
- Циклы
- Переменные
- Таймеры

### Сохранение и экспорт
- Автосохранение черновиков
- Экспорт в JSON
- Импорт готовых шаблонов

## Примеры использования

### Чат-бот поддержки
1. Приветствие пользователя
2. Классификация вопроса
3. Поиск в базе знаний
4. Генерация ответа
5. Передача оператору при необходимости

### Автоматизация продаж
1. Захват лида
2. Квалификация
3. Предложение продукта
4. Обработка возражений
5. Запись в CRM
