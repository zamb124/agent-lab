# Начало работы

Этот раздел поможет вам быстро начать работу с Agent Lab и создать вашего первого агента.

## Шаг 1: Доступ к платформе

### Вход в систему

1. Откройте Agent Lab в браузере
2. Нажмите **"Войти через Яндекс"** (или другой доступный провайдер)
3. Авторизуйтесь через OAuth

### Создание компании

После первого входа вам нужно создать компанию (рабочее пространство):

1. Введите **название компании** (например, "Моя компания")
2. Укажите **поддомен** (например, `mycompany` → `mycompany.agents-lab.ru`)
3. Нажмите **"Создать"**

!!! tip "Совет"
    Поддомен используется для изоляции данных. Каждая компания работает на своем поддомене.

## Шаг 2: Обзор интерфейса

После входа вы увидите главную панель с разделами:

### 🤖 Боты
Управление вашими ботами:
- Список активных ботов
- Создание новых ботов
- Настройка каналов (Telegram, WhatsApp)

### 💬 Чаты
История всех диалогов с пользователями:
- Просмотр переписок
- Анализ запросов
- Отладка сценариев

### 🏪 Store
Магазин готовых агентов и сценариев (в разработке)

### 📜 История
Полная история взаимодействий:
- Все запросы к агентам
- Расход токенов
- Ошибки и предупреждения

### 🎨 Builder
Визуальный конструктор агентов:
- Drag-and-drop интерфейс
- Создание flows без кода
- Настройка логики

### 🔑 Ключи и Переменные
Настройка интеграций:
- API ключи для LLM провайдеров (OpenAI, YandexGPT)
- Токены для мессенджеров
- Переменные окружения

### 💳 Биллинг
Отслеживание расходов:
- Использование токенов
- Вызовы API
- Лимиты и балансы

## Шаг 3: Настройка LLM провайдера

Перед созданием агентов нужно настроить хотя бы один LLM провайдер:

### OpenAI (рекомендуется для начала)

1. Перейдите в **Ключи и Переменные**
2. Найдите переменную `OPENAI_API_KEY`
3. Введите ваш API ключ от OpenAI
4. Сохраните

### YandexGPT

1. Перейдите в **Ключи и Переменные**
2. Настройте переменные:
   - `YANDEX_FOLDER_ID`
   - `YANDEX_SERVICE_ACCOUNT_KEY`
3. Сохраните

!!! info "Где взять API ключи?"
    - **OpenAI**: [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
    - **YandexGPT**: [cloud.yandex.ru/docs/iam/operations/sa/create-access-key](https://cloud.yandex.ru/docs/iam/operations/sa/create-access-key)

## Шаг 4: Создание первого бота

### Через готовые сценарии

Самый простой способ - использовать готовый агент:

1. Перейдите в раздел **Боты**
2. Нажмите **"Создать бота"**
3. Выберите готовый сценарий (например, "FAQ Bot")
4. Введите название бота
5. Нажмите **"Создать"**

Готово! Бот создан и можно его тестировать через веб-интерфейс.

### Подключение Telegram

Чтобы бот работал в Telegram:

1. Создайте бота через [@BotFather](https://t.me/botfather)
2. Получите токен (например: `123456789:ABCdef...`)
3. В Agent Lab перейдите в настройки бота
4. В разделе **Telegram** вставьте токен
5. Нажмите **"Подключить"**

Теперь напишите боту в Telegram - он ответит! 🎉

## Шаг 5: Тестирование

### Веб-виджет чата

В правом нижнем углу любой страницы Agent Lab есть виджет чата:

1. Нажмите на иконку чата
2. Выберите вашего бота
3. Напишите сообщение
4. Получите ответ

### Telegram/WhatsApp

Если вы подключили мессенджер, просто напишите боту!

### История диалогов

Все диалоги сохраняются в разделе **Чаты**:
- Просмотр полной переписки
- Анализ ответов агента
- Отладка проблем
