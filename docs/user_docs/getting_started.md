# Начало работы с Agent Lab

Этот гайд поможет вам за 15 минут начать работу с платформой и создать первого бота. Мы разберем весь процесс от регистрации до запуска в Telegram.

## 🎯 Цели этого гайда

После выполнения всех шагов у вас будет:
- ✅ Аккаунт в Agent Lab
- ✅ Собственная компания (рабочее пространство)
- ✅ Установленный готовый бот из магазина
- ✅ Настроенный Telegram бот
- ✅ Работающий ИИ-помощник

## Шаг 1: Регистрация и вход {#шаг-1-доступ-к-платформе}

### 1.1 Открываем платформу

Перейдите по адресу: `https://agents-lab.ru` или ваш поддомен.

### 1.2 Выбор провайдера

На странице входа выберите удобный способ авторизации:

*[Скриншот страницы входа будет добавлен позже]*

- **Yandex** - для пользователей Яндекса
- **Google** - для пользователей Google

!!! info "Безопасность"
    Мы используем OAuth 2.0 - ваши пароли не хранятся на наших серверах.

### 1.3 Создание компании {#создание-компании}

После успешной авторизации вам нужно создать компанию:

[Скриншот будет добавлен позже]

**Поля для заполнения:**
- **Название компании** - отображаемое имя (например, "Моя компания")
- **Slug компании** - уникальный идентификатор (например, `mycompany`)

!!! warning "Slug правила"
    - Только латинские буквы и цифры
    - Без пробелов и спецсимволов
    - От 3 до 20 символов
    - Пример: `mycompany` → `mycompany.agents-lab.ru`

После создания вы попадете в дашборд компании.

## Шаг 3: Выбор модели ИИ

Agent Lab предоставляет доступ ко всем популярным ИИ-моделям. Вам достаточно просто выбрать подходящую модель - никаких дополнительных настроек не требуется.

### 3.1 Доступные модели

При создании бота вы можете выбрать из следующих моделей:

#### 🤖 Anthropic Claude (рекомендуется)
- **Claude Sonnet 4.5** - самая мощная модель для сложных задач
- **Claude Sonnet 4** - отличная альтернатива предыдущей версии

#### 🤖 Google Gemini
- **Gemini 2.5 Pro** - огромный контекст, отличное качество
- **Gemini 2.5 Flash** - быстрая версия для повседневных задач
- **Gemini 2.0 Flash** - самая быстрая и экономичная

#### 🤖 xAI Grok
- **Grok Code Fast** - специализирована на программировании
- **Grok 4 Fast** - универсальная быстрая модель

#### 🤖 OpenAI GPT
- **GPT-4o** - оптимизированная версия GPT-4

### 3.2 Сравнение моделей

| Модель | Скорость | Качество | Стоимость |
|--------|----------|----------|-----------|
| Claude Sonnet 4.5 | Средняя | Отличное | Средняя |
| Gemini 2.5 Flash | Высокая | Хорошее | Низкая |
| Grok Code Fast | Высокая | Хорошее | Низкая |
| GPT-4o | Средняя | Отличное | Высокая |

!!! tip "Рекомендация"
    Для начала выберите **Claude Sonnet 4.5** - это универсальная модель с отличным качеством ответов.

## Шаг 5: Подключение Telegram

Чтобы бот работал в Telegram, нужно получить токен у BotFather.

### 5.1 Создание бота в Telegram

1. **Напишите @BotFather** в Telegram
2. **Отправьте** `/newbot`
3. **Введите имя** бота (например, "Мой помощник")
4. **Введите username** (должен заканчиваться на `bot`)
5. **Скопируйте токен**

[Скриншот будет добавлен позже]

### 5.2 Настройка в Agent Lab

1. **Перейдите в "Боты"** и откройте вашего бота
2. **Найдите раздел "Платформы"**
3. **Включите Telegram**
4. **Вставьте токен** и нажмите **"Сохранить"**

[Скриншот будет добавлен позже]

### 5.3 Тестирование

**Напишите боту в Telegram** - он должен ответить! 🎉

[Скриншот будет добавлен позже]
