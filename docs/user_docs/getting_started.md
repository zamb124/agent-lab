# Начало работы с Agent Lab :material-play-circle-outline:

Этот гайд поможет вам за 15 минут начать работу с платформой и создать первого бота. Мы разберем весь процесс от регистрации до запуска в Telegram.

## :material-target: Цели этого гайда

После выполнения всех шагов у вас будет:
- ✅ Аккаунт в Agent Lab
- ✅ Собственная компания (рабочее пространство)
- ✅ Установленный готовый бот из магазина
- ✅ Настроенный Telegram бот
- ✅ Работающий ИИ-помощник

## :material-account-plus-outline: Шаг 1: Регистрация и вход

### :material-monitor-dashboard: 1.1 Открываем платформу

Перейдите по адресу: `https://agents-lab.ru` или ваш поддомен.

### :material-account-box-multiple-outline: 1.2 Выбор провайдера

На странице входа выберите удобный способ авторизации:

*[Скриншот страницы входа будет добавлен позже]*

- **Yandex** - для пользователей Яндекса
- **Google** - для пользователей Google

!!! info "Безопасность"
    Мы используем OAuth 2.0 - ваши пароли не хранятся на наших серверах.

### :material-domain: 1.3 Создание компании

После успешной авторизации вам нужно создать компанию:

<div class="video-container">
    <video controls width="100%">
        <source src="../../video/create_company_ru.mov" type="video/mp4">
        Ваш браузер не поддерживает видео HTML5.
    </video>
</div>

**Поля для заполнения:**
- **Название компании** - отображаемое имя (например, "Моя компания")
- **Slug компании** - уникальный идентификатор (например, `mycompany`)

!!! warning "Slug правила"
    - Только латинские буквы и цифры
    - Без пробелов и спецсимволов
    - От 3 до 20 символов
    - Пример: `mycompany` → `mycompany.agents-lab.ru`

После создания вы попадете в дашборд компании.

## :material-view-dashboard-outline: Шаг 2: Знакомство с дашбордом

После создания компании вы увидите главный интерфейс со следующими разделами:

### :material-robot-outline: Основные разделы

#### :material-robot-outline: Боты
Управляйте всеми вашими ботами. Здесь вы можете создавать новых ботов, просматривать существующие, настраивать их параметры, такие как модель ИИ, промпт, таймауты, количество повторных попыток. Также в этом разделе настраиваются платформы для каждого бота.

#### :material-chat-outline: Чаты
Просматривайте историю всех диалогов пользователей с вашими ботами. Доступен поиск по сообщениям, фильтрация по платформам, статусам и сценариям. Вы можете открыть любой чат для детального просмотра переписки.

#### :material-store: Store
Магазин готовых решений, где вы можете установить ботов для типовых задач (поддержка, продажи, FAQ и т.д.) всего в один клик.

#### :material-clock-outline: История
Подробная статистика по всем запросам к ИИ, включая расход токенов, стоимость, время выполнения и информацию об ошибках. Позволяет отслеживать производительность и затраты.

#### :material-sitemap-outline: Builder
Визуальный конструктор для создания сложных сценариев взаимодействия ботов. Здесь вы можете проектировать логику flow с помощью Drag-and-drop интерфейса, соединять агентов и инструменты, а также настраивать их поведение без необходимости написания кода.

#### :material-key-outline: Ключи и Переменные
Раздел для управления API ключами (например, для OpenAI, YandexGPT) и токенами для интеграций с платформами (Telegram, WhatsApp). Здесь также можно определять переменные окружения, которые будут использоваться вашими ботами.

#### :material-currency-usd: Биллинг
Управление расходами на платформе. Здесь отображается текущий баланс, лимиты, история платежей и информация о пополнении счета.

## :material-brain: Шаг 3: Выбор модели ИИ

Agent Lab предоставляет доступ ко всем популярным ИИ-моделям. Вам достаточно просто выбрать подходящую модель - никаких дополнительных настроек не требуется.

### :material-check-circle-outline: 3.1 Доступные модели

При создании бота вы можете выбрать из следующих моделей:

#### :material-alpha-c-circle-outline: Anthropic Claude (рекомендуется)
- **Claude Sonnet 4.5** - самая мощная модель для сложных задач
- **Claude Sonnet 4** - отличная альтернатива предыдущей версии

#### :material-google: Google Gemini
- **Gemini 2.5 Pro** - огромный контекст, отличное качество
- **Gemini 2.5 Flash** - быстрая версия для повседневных задач
- **Gemini 2.0 Flash** - самая быстрая и экономичная

#### :material-star-outline: xAI Grok
- **Grok Code Fast** - специализирована на программировании
- **Grok 4 Fast** - универсальная быстрая модель

#### :material-openai: OpenAI GPT
- **GPT-4o** - оптимизированная версия GPT-4

### :material-compare-horizontal: 3.2 Сравнение моделей

| Модель | Скорость | Качество | Стоимость |
|--------|----------|----------|-----------|
| Claude Sonnet 4.5 | Средняя | Отличное | Средняя |
| Gemini 2.5 Flash | Высокая | Хорошее | Низкая |

## :material-store-plus-outline: Шаг 4: Установка готового бота из Store

Самый быстрый способ начать - взять готового бота из магазина.

### :material-store-search-outline: 4.1 Открываем Store

В левом меню нажмите **"Store"** (магазин).

### :material-robot-happy-outline: 4.2 Выбираем бота

Найдите подходящего бота. Для начала подойдет **"FAQ Bot"** или **"Support Bot"**.

### :material-download-box-outline: 4.3 Устанавливаем бота

1. **Нажмите на карточку** бота
2. **Посмотрите описание** и возможности
3. **Нажмите "Установить"**

### :material-check-circle-outline: 4.4 Проверяем установку

После установки бот появится в разделе **"Боты"**.

## :material-connection: Шаг 5: Подключение платформ

Agent Lab поддерживает следующие платформы для ваших ботов:
- **Telegram**: для взаимодействия через мессенджер Telegram.
- **WhatsApp**: для интеграции с WhatsApp Business API.
- **Web**: для встраивания чат-бота на ваш веб-сайт.
- **API**: для интеграции бота с вашими системами через REST API.
- **AmoCRM**: для автоматизации работы с клиентами в AmoCRM.

### :material-telegram: 5.1 Подключение Telegram

Чтобы бот работал в Telegram, нужно получить токен у BotFather.

1. **Напишите @BotFather** в Telegram
2. **Отправьте** `/newbot`
3. **Введите имя** бота (например, "Мой помощник")
4. **Введите username** (должен заканчиваться на `bot`)
5. **Скопируйте токен**

### :material-cog-outline: 5.2 Настройка в Agent Lab

1. **Перейдите в "Боты"** и откройте вашего бота
2. **Найдите раздел "Платформы"**
3. **Включите Telegram**
4. **Вставьте токен** и нажмите **"Сохранить"**

### :material-check-circle-outline: 5.3 Тестирование

**Напишите боту в Telegram** - он должен ответить! 🎉

## :material-test-tube: Шаг 6: Тестирование и отладка

### :material-web: 6.1 Веб-чат виджет

В правом нижнем углу интерфейса есть виджет для тестирования:

1. **Нажмите на иконку** чата
2. **Выберите бота** из списка
3. **Напишите сообщение**
4. **Получите ответ** от ИИ

### :material-chat-outline: 6.2 Просмотр истории

Все диалоги сохраняются в разделе **"Чаты"**:

- **Полная переписка** - все сообщения
- **Поиск** по тексту
- **Аналитика** - время ответа, ошибки

### :material-chart-bar: 6.3 Статистика использования

В разделе **"История"** отслеживайте:

- **Расход токенов** - стоимость запросов
- **Время выполнения** - производительность
- **Ошибки** - для отладки
