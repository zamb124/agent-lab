В левом меню нажмите **"Боты"**:

![Раздел Боты](img/bots_menu.png)

Нажмите **"+ Создать бота"** или кнопку **"Создать бота"**:

![Кнопка создания бота](img/bots_create_button.png)

### Шаг 3: Заполняем основные поля

#### Название и описание

![Форма создания бота](img/bots_create_form.png)

!!! info "Entry Point Agent"
    Это главный "мозг" бота. Он обрабатывает все входящие сообщения и решает, что делать.

## 🤖 Создание Entry Point Agent

### Что такое агент

Агент - это ИИ с инструкциями и инструментами:

- **Промпт** - инструкции для ИИ
- **Инструменты** - что умеет делать бот
- **Переменные** - настройки и данные
- **Модель ИИ** - GPT-4, YandexGPT и др.

### Создание простого агента

#### Шаг 1: Выбираем модель ИИ

![Выбор модели ИИ](img/agent_model_select.png)

**Рекомендации:**
- **Claude Sonnet 4.5** - универсальная модель высокого качества
- **Gemini 2.5 Flash** - быстрая и экономичная
- **Grok Code Fast** - для задач программирования

#### Шаг 2: Пишем промпт

Это самая важная часть! Промпт определяет поведение бота.

![Редактор промпта](img/agent_prompt_editor.png)

**Пример простого промпта:**

```
Ты - помощник компании "Магазин Техники".
Твоя задача:
- Отвечать на вопросы о товарах
- Помогать с выбором товаров
- Принимать заказы
- Быть вежливым и полезным

Информация о компании:
- Мы продаем электронику и бытовую технику
- Доставка по России за 1-3 дня
- Гарантия на все товары 1 год
```

**Пример продвинутого промпта:**

```
Ты - эксперт по продаже смартфонов в компании "ТехноМаг".

Правила общения:
- Всегда здоровайся при первом сообщении
- Используй имя клиента, если известно
- Предлагай 2-3 варианта вместо одного
- Упомянай преимущества товаров
- Спрашивай о бюджете и предпочтениях

Структура ответа:
1. Приветствие и уточнение потребностей
2. Предложение вариантов
3. Сравнение характеристик
4. Информация о цене и наличии
5. Предложение помощи с оформлением

Важно:
- Не дави на покупку
- Будь честен о недостатках
- Предлагай аксессуары
- Напоминай о гарантиях
```

#### Шаг 3: Добавляем переменные

Переменные позволяют персонализировать бота:

![Переменные агента](img/agent_variables.png)

Примеры переменных:
- company_name - название компании
- working_hours - время работы
- delivery_time - сроки доставки
- support_phone - телефон поддержки
- return_policy - условия возврата

В промпте используйте переменные:
```
Работаем {{ working_hours }}
Доставка {{ delivery_time }}
```

### Добавление инструментов (Tools)

Инструменты позволяют боту взаимодействовать с внешними системами.

#### Шаг 1: Выбираем инструменты

![Выбор инструментов](img/agent_tools_select.png)

#### Популярные инструменты

##### Веб-поиск
- **web_search** - поиск в интернете
- **website_content** - чтение страниц сайтов

##### Работа с данными
- **database_query** - запросы к базам данных
- **api_call** - вызовы внешних API

##### Генерация контента
- **image_generate** - создание изображений
- **text_summarize** - суммаризация текста

##### Сессионные инструменты
- **session_set** - сохранение данных
- **session_get** - получение данных

#### Шаг 2: Настройка инструментов

Для каждого инструмента можно настроить параметры:

![Настройка инструмента](img/tool_configuration.png)

**Пример настройки поиска:**
```json
{
  "max_results": 5,
  "language": "ru",
  "safe_search": true
}
```

## ⚙️ Настройка бота

### Глобальные переменные

Переменные доступные всем агентам бота:

![Глобальные переменные](img/bot_global_variables.png)

Примеры глобальных переменных:
- api_url - URL внешнего API
- database_url - строка подключения к БД
- smtp_settings - настройки email

### Дополнительные настройки

- **Timeout** - максимальное время ожидания ответа
- **Max Retries** - количество попыток при ошибке
- **Temperature** - уровень креативности ответов

### RAG (База знаний)

Подключение документов для улучшения ответов:

![Настройки RAG](img/bot_rag_settings.png)

- **Включить RAG** - да/нет
- **Область поиска** - flow/company
- **Автоиндексация** - индексировать новые сообщения

## 🌐 Подключение платформ

### Telegram

#### Шаг 1: Создание бота
1. Напишите **@BotFather** в Telegram
2. Отправьте `/newbot`
3. Введите имя и username
4. **Скопируйте токен**

#### Шаг 2: Настройка в Agent Lab

![Настройка Telegram](img/bot_telegram_setup.png)

1. **Включите Telegram**
2. **Вставьте токен**
3. **Сохраните**

!!! success "Готово!"
    Бот заработает через 30 секунд.

### WhatsApp Business

#### Шаг 1: Регистрация в Meta
1. Создайте аккаунт в [Business Manager](https://business.facebook.com)
2. Зарегистрируйте приложение WhatsApp
3. Получите **Phone Number ID** и **Access Token**

#### Шаг 2: Настройка webhook
1. В Meta настройте webhook URL:
   `https://{company_subdomain}.agents-lab.ru/api/v1/webhook/whatsapp/{bot_id}`
2. Укажите **Verify Token**

#### Шаг 3: В Agent Lab

![Настройка WhatsApp](img/bot_whatsapp_setup.png)

1. **Включите WhatsApp**
2. **Введите Phone Number ID**
3. **Введите Access Token**
4. **Укажите Verify Token**

### Web Chat виджет

#### Шаг 1: Включение виджета

![Настройка Web Chat](img/bot_webchat_setup.png)

1. **Включите Web Chat**
2. **Получите код виджета**

#### Шаг 2: Установка на сайт

Вставьте код перед `</body>`:

```html
<script>
  window.agentLabConfig = {
    botId: 'your_bot_id',
    apiUrl: 'https://{company_subdomain}.agents-lab.ru'
  };
</script>
<script src="https://{company_subdomain}.agents-lab.ru/static/chat-widget.js"></script>
```

## 🚨 Устранение проблем

### Бот не отвечает

**Проверьте:**
- ✅ API ключи настроены
- ✅ Платформы подключены
- ✅ Агент выбран правильно
- ✅ Промпт не пустой

### Неправильные ответы

**Решения:**
- ✏️ Улучшите промпт
- 📚 Добавьте базу знаний
- 🔧 Измените модель ИИ
- 📝 Добавьте примеры

### Медленная работа

**Оптимизации:**
- ⚡ Выберите быструю модель (GPT-3.5)
- 📝 Сократите промпт
- 🔄 Уменьшите temperature
- 🗂️ Ограничьте поиск в RAG
