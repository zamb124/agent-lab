# Устранение проблем

!!! note "В разработке"
    Этот раздел находится в разработке. Скоро здесь появятся решения наиболее частых проблем.

## Общие проблемы

### Бот не отвечает

**Возможные причины:**
- Неправильные API ключи
- Недостаточно средств на балансе
- Проблемы с подключением к платформам

**Решения:**
- Проверьте настройки API ключей
- Проверьте баланс компании
- Посмотрите логи в разделе "История"

### Медленные ответы

**Оптимизации:**
- Выберите более быструю модель ИИ
- Сократите длину промпта
- Проверьте лимиты и квоты

### Ошибки биллинга

**BillingError:**
- Проверьте баланс компании
- Убедитесь, что не превышен месячный лимит

**TariffError:**
- Проверьте доступность ресурса для вашего тарифа
- Обратитесь в поддержку для повышения тарифа

## Проблемы по платформам

### Telegram
- Проверьте токен BotFather
- Убедитесь, что бот не заблокирован
- Проверьте права доступа

### WhatsApp
- Проверьте настройки webhook
- Убедитесь в валидности токенов Meta
- Проверьте статус номера телефона

### Web Chat
- Проверьте HTTPS сертификат
- Убедитесь в корректности кода виджета
- Проверьте настройки CORS
