## Общие вопросы

### Что такое Agent Lab?
Agent Lab - платформа для создания и управления ИИ-агентами без навыков программирования.

### Как начать работу?
1. Зарегистрируйтесь через Yandex или Google
2. Создайте компанию
3. Установите API ключи
4. Создайте первого бота из Store

### Сколько стоит использование?
- **Free**: 1000 запросов в месяц
- **Basic**: 990₽/месяц + оплата за ИИ
- **Premium**: 2990₽/месяц + оплата за ИИ

## Проблемы и решения

### Бот не отвечает
Проверьте:
- Подключены ли платформы
- Настроены ли API ключи
- Правильный ли промпт

### Высокая стоимость
Оптимизируйте:
- Используйте GPT-3.5 для простых задач
- Сокращайте промпты
- Настраивайте кэширование
