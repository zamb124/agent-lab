# Переменные и настройки

!!! note "В разработке"
    Этот раздел находится в разработке. Скоро здесь появится подробная документация по работе с переменными в Agent Lab.

## Что такое переменные

Переменные в Agent Lab позволяют:

- Персонализировать поведение ботов
- Хранить конфиденциальные данные (API ключи, токены)
- Передавать данные между компонентами
- Управлять настройками на уровне компании

## Типы переменных

### Глобальные переменные компании
Доступны всем ботам в компании.

### Системные переменные
Предустановленные переменные платформы.

## Примеры использования

Переменные могут хранить:
- Название компании и контактные данные
- URL внешних сервисов
- Настройки поведения ботов
- Ключи API (но лучше использовать системные переменные)
