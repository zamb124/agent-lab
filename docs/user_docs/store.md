# Магазин ботов (Store)

Магазин Agent Lab - это коллекция готовых ИИ-ботов для решения типовых бизнес-задач. Устанавливайте проверенные решения одним кликом без программирования.

## 🏪 Что такое Store

Store - это каталог готовых к использованию ботов:

- **Одним кликом** - установка занимает секунды
- **Проверенные решения** - протестированы и оптимизированы
- **Разные категории** - поддержка, продажи, HR, знания
- **Регулярные обновления** - новые боты добавляются постоянно

!!! tip "Рекомендация для новичков"
    Начните с Store - это самый быстрый способ получить работающего бота!

## 🚀 Как установить бота из Store

### Шаг 1: Открываем Store

В левом меню нажмите **"Store"**:

![Открываем Store](img/store_menu.png)

### Шаг 2: Выбираем категорию

Используйте фильтры для поиска подходящего бота:

![Фильтры Store](img/store_filters.png)

- **По категории** - поддержка, продажи, HR
- **По платформе** - Telegram, WhatsApp, Web
- **По популярности** - самые устанавливаемые

### Шаг 3: Просматриваем описание

Кликните на карточку бота для детального просмотра:

![Карточка бота](img/store_bot_card.png)

**Что посмотреть:**
- ✅ **Описание** - что умеет бот
- ✅ **Платформы** - где можно использовать
- ✅ **Автор** - кто создал решение
- ✅ **Отзывы** - опыт других пользователей

### Шаг 4: Устанавливаем бота

Нажмите **"Установить"** на странице бота:

![Установка бота](img/store_install_button.png)

**Процесс установки:**
1. **Подтверждение** - нажмите "Установить"
2. **Настройка названия** - придумайте имя для бота
3. **Автоматическая установка** - занимает 5-10 секунд

### Шаг 5: Проверяем результат

После установки бот появится в разделе **"Боты"**:

![Бот установлен](img/store_bot_installed.png)

**Статус:** ✅ **Установлен**

## ⚙️ Настройка установленного бота

### Базовые настройки

1. **Перейдите в "Боты"**
2. **Кликните на карточку** установленного бота
3. **Откройте раздел "Настройки"**

![Настройки бота](img/store_bot_settings.png)

#### Название и описание
- **Название** - отображаемое имя бота
- **Описание** - для чего предназначен

#### Переменные бота
- **Company Name** - название вашей компании
- **Support Email** - email поддержки
- **Working Hours** - время работы

### Подключение платформ

#### Telegram
1. **Получите токен** у [@BotFather](https://t.me/botfather)
2. **Включите Telegram** в настройках бота
3. **Вставьте токен** и сохраните

#### WhatsApp
1. **Зарегистрируйтесь** в WhatsApp Business
2. **Получите API ключ**
3. **Настройте интеграцию**

#### Web-виджет
1. **Включите Web Chat**
2. **Получите код виджета**
3. **Вставьте на сайт**

## 🔧 Кастомизация бота

### Изменение промпта

Большинство ботов позволяют настроить системный промпт:

![Редактирование промпта](img/store_prompt_edit.png)

**Примеры изменений:**
- Добавить информацию о компании
- Изменить стиль общения
- Добавить специфические инструкции

### Добавление переменных

Расширьте возможности через переменные:

Примеры переменных для настройки:
- company_name - название вашей компании
- support_email - email поддержки
- products - список продуктов
- pricing - информация о ценах

### Интеграция с базами знаний

Загрузите документы для улучшения ответов:

1. **Перейдите в настройки RAG**
2. **Включите "База знаний"**
3. **Загрузите документы** (PDF, DOC, TXT)
