# Магазин ботов (Store) :material-store:

Раздел **"Store"** — это коллекция готовых ИИ-ботов (Flows) для решения типовых бизнес-задач. Вы можете устанавливать проверенные решения одним кликом без необходимости программирования и быстро запускать их в работу.

## :material-shopping-outline: Что такое Store

Store предоставляет:
- **Быстрый старт**: Установка бота занимает считанные секунды.
- **Готовые решения**: Коллекция протестированных и оптимизированных ботов для различных задач (поддержка, продажи, HR, база знаний).
- **Простота использования**: Установка и базовая настройка без кода.
- **Регулярные обновления**: Каталог ботов постоянно пополняется и обновляется.

!!! tip "Рекомендация для новичков"
    Начните работу с Agent Lab с установки готового бота из Store — это самый быстрый способ получить работающего ИИ-помощника!

## :material-rocket-launch-outline: Как установить бота из Store

### :material-store-search-outline: Шаг 1: Откройте Store

В левом боковом меню навигации нажмите на пункт **"Store"**.

### :material-selection-multiple: Шаг 2: Выберите бота

В Store вы увидите список доступных ботов, каждый из которых представлен карточкой с названием, описанием, поддерживаемыми платформами и информацией об авторе. Вы можете использовать фильтры для поиска по категориям или платформам.

Нажмите на карточку любого бота, чтобы просмотреть детальную информацию о нем.

### :material-file-document-outline: Шаг 3: Просмотрите детали Flow

В модальном окне с деталями Flow вы найдете:
- **Полное описание** функционала бота.
- Список **поддерживаемых платформ**.
- Информацию об **авторе** Flow.
- Кнопку **"Установить"**.

### :material-download-box-outline: Шаг 4: Установите бота

Нажмите кнопку **"Установить"**. Бот будет автоматически добавлен в ваш список ботов. Если бот уже установлен, кнопка изменится на **"Удалить"**.

### :material-check-circle-outline: Шаг 5: Проверьте установку

После установки перейдите в раздел **"Боты"**. Ваш новый бот появится в списке с отметкой **"Установлено"**.

## :material-cog-outline: Настройка установленного бота

После установки бот из Store готов к работе, но вы можете настроить его под свои нужды.

1. Перейдите в раздел **"Боты"**.
2. Нажмите на карточку установленного бота, чтобы открыть его детальные настройки.
3. В открывшемся окне вы сможете изменить:
   - **Название и описание** бота.
   - **Промпт агента** для изменения его поведения.
   - **Модель ИИ** и параметры (температура, макс. токены).
   - **Базу знаний (RAG)**, загрузить свои документы.
   - **Каналы**, подключить Telegram, WhatsApp, Web Chat, API или AmoCRM.
   - **Переменные** для персонализации ответов.
4. Нажмите **"Сохранить"** для применения изменений.

## :material-delete-empty-outline: Удаление Flow из Store

Если вы хотите удалить установленный Flow:
1. Откройте детали Flow в Store.
2. Нажмите кнопку **"Удалить"**.
3. Подтвердите удаление.

!!! warning "Важно"
    Удаление Flow из Store приведет к его полному удалению из вашей компании. История диалогов с этим ботом сохранится в разделе **"Чаты"**.
