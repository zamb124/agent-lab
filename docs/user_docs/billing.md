# Биллинг и оплата

!!! note "На основе реальной системы"
    Документация основана на действующей системе биллинга Agent Lab.

## Принцип работы

Agent Lab использует модель **pay-as-you-go** - оплата только за фактическое использование ресурсов:

- **LLM запросы** - оплата за токены (входящие/исходящие)
- **Инструменты** - оплата за вызовы
- **Баланс компании** - предоплаченные средства

## Тарифные планы

### FREE
- **Множитель стоимости: 1.0** (полная цена)
- Базовые модели ИИ доступны
- Все платформы поддерживаются

### BASIC
- **Множитель стоимости: 0.7-0.8** (скидка 20-30%)
- Доступ к продвинутым моделям
- Приоритетная техническая поддержка

### PREMIUM
- **Множитель стоимости: 0.3-0.5** (скидка 50-70%)
- Максимальные скидки на все ресурсы
- Расширенная поддержка

### ENTERPRISE
- **Множитель стоимости: 0.0** (бесплатно)
- Полный доступ ко всем ресурсам
- Индивидуальные условия

## Стоимость ресурсов

### LLM модели (за токен)

| Модель | Input токен | Output токен |
|--------|-------------|--------------|
| Claude Sonnet 4.5 | 0.00003₽ | 0.00015₽ |
| Claude Opus 4 | 0.00015₽ | 0.00075₽ |
| GPT-4o | 0.000225₽ | 0.0009₽ |

### Инструменты (за вызов)

| Инструмент | Стоимость |
|------------|-----------|
| Weather API | 0.1₽ |
| Travel Suggest | 0.2₽ |
| Calculator | 0₽ (бесплатно) |
| NanoBanana Generation | 0.5₽ |
| FASHN Buyer | 0₽ (бесплатно) |

## Примеры расчета стоимости

### FREE план, Claude Sonnet 4.5
- Входящие: 1000 токенов × 0.00003₽ = 0.03₽
- Исходящие: 500 токенов × 0.00015₽ = 0.075₽
- **Итого: 0.105₽**

### BASIC план, тот же запрос
- Базовая стоимость: 0.105₽
- Множитель: 0.8 (скидка 20%)
- **Итого: 0.084₽**

### PREMIUM план, Weather API
- Базовая стоимость: 0.1₽
- Множитель: 0.3 (скидка 70%)
- **Итого: 0.03₽**

## Управление балансом

### Текущий баланс
Каждая компания имеет предоплаченный баланс в рублях.

### Месячный лимит (опционально)
Можно установить лимит расходов на месяц для контроля затрат.

### Статистика использования
Отслеживание расходов по:
- Типам ресурсов (LLM, инструменты)
- Пользователям
- Периодам времени

## Мониторинг расходов

### Просмотр статистики
В интерфейсе доступна статистика использования по:
- Общая стоимость за период
- Количество запросов
- Расходы по типам ресурсов
- Использование по пользователям

## Ограничения и проверки

### Перед использованием
Система автоматически проверяет:
- Достаточно ли средств на балансе
- Не превышен ли месячный лимит (если установлен)

### Ограничения использования
Система автоматически контролирует:
- Достаточно ли средств на балансе
- Соблюдение месячных лимитов (если установлены)

### При проблемах
- Проверьте баланс компании
- Убедитесь, что не превышен лимит
- Обратитесь в поддержку при необходимости
