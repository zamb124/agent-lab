# Биллинг и оплата :material-currency-usd:

Раздел **"Биллинг"** в Agent Lab предназначен для мониторинга и управления расходами на использование платформы. Здесь вы можете отслеживать потребление ресурсов, пополнять баланс и контролировать затраты.

## :material-credit-card-outline: Принцип работы

Agent Lab использует модель **pay-as-you-go** (плати по факту использования) — вы оплачиваете только фактически потребленные ресурсы:
- **LLM запросы**: Оплата за токены, используемые ИИ-моделями (как входные, так и выходные).
- **Вызовы инструментов**: Оплата за использование специализированных инструментов (например, Weather API, NanoBanana Generation).
- **Баланс компании**: Ваши средства хранятся на предоплаченном балансе компании.

## :material-cash-multiple: Тарифные планы

### :material-account-box-outline: FREE
- **Множитель стоимости: 1.0** (полная цена за ресурсы).
- Доступ к базовым моделям ИИ.
- Поддержка всех платформ.

### :material-account-tie-outline: BASIC
- **Множитель стоимости: 0.7-0.8** (скидка 20-30% на стоимость ресурсов).
- Доступ к продвинутым моделям.
- Приоритетная техническая поддержка.

### :material-account-star-outline: PREMIUM
- **Множитель стоимости: 0.3-0.5** (скидка 50-70% на стоимость ресурсов).
- Максимальные скидки на все ресурсы.
- Расширенная поддержка.

### :material-bank-outline: ENTERPRISE
- **Множитель стоимости: 0.0** (бесплатно).
- Полный доступ ко всем ресурсам и индивидуальные условия обслуживания.

## :material-wallet-outline: Управление балансом

### :material-cash-check: Текущий баланс
Каждая компания имеет предоплаченный баланс в рублях, который расходуется при использовании ресурсов.

### :material-cash-plus: Пополнение баланса
В разделе "Биллинг" вы найдете опции для пополнения баланса компании.

### :material-chart-timeline-variant: Месячный лимит
Вы можете установить опциональный месячный лимит расходов для каждого бота или для всей компании, чтобы контролировать затраты.

## :material-chart-bar: Мониторинг расходов

В разделе "Биллинг" доступна подробная статистика использования, позволяющая отслеживать:
- Общую стоимость за выбранный период.
- Расходы по типам ресурсов (LLM, инструменты).
- Количество запросов и вызовов.
- Использование ресурсов по отдельным ботам или пользователям.

## :material-shield-keyhole-outline: Ограничения и проверки

Система Agent Lab автоматически проверяет наличие достаточных средств на балансе и соблюдение установленных месячных лимитов перед выполнением каждого запроса или вызова инструмента. В случае превышения лимитов или недостатка средств, запрос будет отклонен.

## :material-arrow-down-drop-circle-outline: Оптимизация затрат

Для снижения расходов на платформе рекомендуется:
1. **Выбирать подходящие LLM модели**: Используйте более экономичные модели (например, Flash версии) для простых задач и более дорогие только при необходимости.
2. **Оптимизировать промпты**: Сокращайте длину входных и выходных промптов, чтобы уменьшить количество потребляемых токенов.
3. **Использовать кэширование**: Для повторяющихся запросов, если применимо.
4. **Устанавливать лимиты**: Активно используйте месячные лимиты для контроля бюджета.
5. **Мониторить статистику**: Регулярно просматривайте раздел "Биллинг" для выявления аномалий и возможностей для оптимизации.
