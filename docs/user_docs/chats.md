# Чаты

Раздел **"Чаты"** в Agent Lab предоставляет полный доступ к истории всех диалогов пользователей с вашими ботами. Это централизованное место для просмотра, анализа и мониторинга взаимодействия.

## 📈 Обзор раздела "Чаты"

В разделе "Чаты" вы можете:
- Просматривать список всех активных и завершенных сессий.
- Фильтровать диалоги по различным критериям.
- Открывать любую сессию для просмотра полной переписки.
- Анализировать ключевые метрики взаимодействия.

## 🔍 Просмотр списка сессий

При открытии раздела "Чаты" вы увидите таблицу со списком сессий. Каждая строка в таблице представляет отдельный диалог и содержит следующую информацию:

- **Канал**: Канал, через который происходит общение (Telegram, WhatsApp, Web, API, AmoCRM).
- **Flow**: Название Flow (бота), с которым велся диалог.
- **Первое сообщение**: Начало диалога с пользователем.
- **Статус**: Текущий статус сессии (например, `active`, `inactive`, `processing`, `waiting_input`).
- **Сообщений**: Количество сообщений в данной сессии.
- **Создано**: Дата и время начала сессии.
- **Активность**: Дата и время последнего сообщения в сессии.
- **Чат**: Кнопка для быстрого перехода в интерактивный чат с ботом.

## 🛠️ Фильтрация сессий

Вы можете использовать фильтры в верхней части страницы для быстрого поиска нужных диалогов:

- **Канал**: Выберите конкретный канал (например, Telegram, Web) для отображения только диалогов с него.
- **Flow**: Введите часть названия или ID Flow для фильтрации по ботам.
- **Статус**: Фильтруйте по статусу сессии.
- **Дата От / До**: Укажите временной диапазон для поиска диалогов, начатых в определенный период.

Нажмите кнопку **"Применить"** для активации фильтров или **"Сбросить"** для очистки.

## 💬 Просмотр деталей диалога

Чтобы просмотреть полную историю сообщений в конкретной сессии:

1. В таблице сессий найдите нужный диалог.
2. Нажмите кнопку **"Чат"** (иконка облачка с точками) в столбце **"Чат"**.
3. Откроется модальное окно с подробной перепиской, где будут видны сообщения пользователя, ответы бота, вызовы инструментов и системные события.

## 📊 Аналитика диалогов

Анализ диалогов в разделе "Чаты" позволяет вам:
- Понять, какие вопросы задают пользователи.
- Оценить качество ответов бота.
- Выявить области для улучшения промптов и Flow.
- Мониторить общую активность бота.
