# Документация API :material-api:

Здесь вы можете найти подробную документацию по REST API платформы Agent Lab.

## :material-api-rest: Swagger UI

**[Открыть Swagger UI](http://localhost:8001/api/docs)**

Интерактивная документация API, позволяющая просматривать все доступные endpoint, их параметры, модели данных и тестировать запросы прямо из браузера.
