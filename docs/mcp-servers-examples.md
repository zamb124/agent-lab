# Публичные MCP серверы (HTTP/SSE)

## Реально работающие публичные MCP серверы

| Сервер | URL | Транспорт | API ключ |
|--------|-----|-----------|----------|
| Context7 (официальный) | `https://mcp.context7.com/mcp` | HTTP | Да (бесплатный на context7.com/dashboard) |
| Context7 HTTP (неофициальный) | `https://context7.liam.sh/mcp` | HTTP | Нет |
| Context7 HTTP (неофициальный) | `https://context7.liam.sh/sse` | SSE | Нет |
