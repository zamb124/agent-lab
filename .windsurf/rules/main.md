# Правила чтения документации

При работе с проектом всегда обращайся к документации:

- `README.md` в корне - общие сведения о проекте
- `docs/` - полная документация
- `docs/architecture.md` - архитектура системы
- `docs/api.md` - API документация
- `docs/frontend.md` - фронтенд архитектура

Используй Context7 для получения актуальной документации LangGraph:
- https://langchain-ai.github.io/langgraph/
