# Документация в коде: as is

Комментарии, cursor rules, docstrings, `description` в схемах и подписи в UI — **только как устроено сейчас** (контракт, семантика, зона ответственности).

**Не писать** в них: прошлое поведение, ход миграций/рефакторинга, «раньше / теперь / вместо». История — коммиты, PR, CHANGELOG, `migrations.mdc`, не inline.

Поведение изменилось — **переписать** текст целиком, не дописывать «обновлено…». Устаревший текст — править вместе с кодом.
