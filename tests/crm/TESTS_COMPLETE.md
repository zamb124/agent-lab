## Как запустить

### Быстрый старт

```bash

# С покрытием

```

## Использованные фикстуры

**Из `tests/conftest.py`:**
- `app` - FastAPI приложение
- `unique_id` - изоляция тестов
- `mock_llm_redis` - mock LLM
- `client` - HTTP клиент
- `taskiq_worker` - TaskIQ worker
- `rag_worker` - RAG worker

**Из `tests/crm/conftest.py`:**
- `crm_client` - HTTP клиент для CRM API
- `crm_container` - CRM container

## Принципы тестирования

1. **E2E тесты** - проверяем полные user journeys через API
2. **Без моков** - используем реальные сервисы (кроме MockLLM)
3. **Изоляция** - каждый тест использует `unique_id`
4. **Параллельность** - тесты можно запускать параллельно
5. **Читаемость** - тесты как живая документация
