HumanitecAgent

AI-агент на вашем компьютере.
Работает с файлами, запускает программы, подключается к Humanitec Platform.

📁

Локальные файлы

Читайте и редактируйте файлы на компьютере. Агент видит только то, что вы разрешили.

💻

Shell и программы

Запускайте bash, Python, git и другие инструменты. С whitelist и confirm-диалогами.

🌐

Platform flows

Все сценарии Humanitec доступны как MCP tools прямо в чате агента.

🔒

Приватность

Данные остаются на вашем компьютере. Агент работает локально, audit-логи на платформе.

Другие платформы
  • Windows (.msi)
  • macOS Apple Silicon (.dmg)
  • macOS Intel (.dmg)
  • Linux Debian/Ubuntu (.deb)
  • Linux Fedora/RHEL (.rpm)
  • Linux (AppImage)
