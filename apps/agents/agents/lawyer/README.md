# Юридический агент ЭНЖИЛАБС

Юридический агент для компании ЭНЖИЛАБС, специализирующийся на российском законодательстве и юридическом сопровождении.

## Описание

**LawyerAgent** — профессиональный юридический советник, который:

- Консультирует по российскому законодательству (ГК РФ, НК РФ, ТК РФ, КоАП РФ, УК РФ)
- Работает с корпоративным, договорным, трудовым и налоговым правом
- Имеет доступ к базе знаний компании с документами
- Может загружать и анализировать юридические документы
- Использует модель `google/gemini-2.5-pro` для глубокого анализа

## Информация о компании

Агент работает с данными компании:

- **Краткое наименование**: ООО ЭНЖИЛАБС
- **Полное наименование**: Общество с Ограниченной Ответственностью ЭНЖИЛАБС
- **Английское наименование**: Limited Liability Company Angilabs (Angilabs LLC)
- **Генеральный директор**: Шведов Виктор Викторович

## Возможности

### Инструменты

1. **search_knowledge_base** — поиск информации в базе знаний компании
2. **upload_document_to_knowledge_base** — загрузка документов в базу знаний
3. **list_documents_in_knowledge_base** — просмотр всех документов
4. **read_file** — чтение содержимого файла
5. **ask_user** — запрос дополнительной информации у пользователя

### RAG (Retrieval-Augmented Generation)

Агент имеет доступ к RAG системе с двумя уровнями поиска:
- **company** — документы общие для всей компании ЭНЖИЛАБС
- **flow** — документы специфичные для юридического агента

## Использование

### Через API

```bash
curl -X POST http://localhost:8001/api/v1/flows/app.flows.lawyer_flow.lawyer_flow/message \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Какие налоговые риски для ООО при работе с физлицами без самозанятости?"
  }'
```

### Через Telegram

Если настроен токен бота в переменных:
- `lawyer_telegram_username` — username бота
- `lawyer_telegram_token` — токен бота

### Примеры запросов

1. **Консультация по договору**:
   > "Проанализируй договор поставки. Какие риски для ООО ЭНЖИЛАБС?"

2. **Поиск в базе знаний**:
   > "Найди договоры с ООО Поставщик за 2024 год"

3. **Загрузка документа**:
   > (Прикрепить файл) "Загрузи этот договор в базу знаний"

4. **Трудовое право**:
   > "Как правильно оформить сокращение штата для ООО?"

5. **Налоговое право**:
   > "Какие налоговые вычеты доступны для ООО в 2025 году?"

## Настройка

### Конфигурация Flow

Настройки находятся в `app/flows/lawyer_flow.py`:

```python
variables={
    "company_short_name": "ООО ЭНЖИЛАБС",
    "company_full_name": "Общество с Ограниченной Ответственностью ЭНЖИЛАБС",
    "company_short_name_en": "Angilabs LLC",
    "company_full_name_en": "Limited Liability Company Angilabs",
    "ceo_name": "Шведов Виктор Викторович",
}

llm_config={
    "model": "google/gemini-2.5-pro",
    "temperature": 0.2,
    "max_tokens": 8192
}

rag_config={
    "enabled": True,
    "namespace_scope": "company",
    "search_scopes": ["company", "flow"],
    "auto_index_messages": False
}
```

### Подключение Telegram бота

Добавьте в конфигурацию переменные:

```json
{
  "lawyer_telegram_username": "@your_lawyer_bot",
  "lawyer_telegram_token": "YOUR_BOT_TOKEN"
}
```

## Рабочий процесс агента

1. **Анализ запроса** — понимание юридического вопроса
2. **Поиск в базе** — проверка документов компании
3. **Правовой анализ** — применение знаний законодательства РФ
4. **Формирование ответа** — структурированный юридически корректный ответ
5. **Рекомендации** — конкретные действия и решения

## Примечания

- Агент всегда ссылается на конкретные статьи законов РФ
- Учитывает специфику компании ЭНЖИЛАБС
- При неясности уточняет детали у пользователя
- Сохраняет важные документы в базу знаний
- Использует профессиональную юридическую терминологию
