🧪 Тест встроенного чата

Инструкция:
  1. Сначала получите токен через API или используйте токен из демо-страницы
  2. Введите URL вашего сервера Agent Lab
  3. Введите токен и ID агента
  4. Нажмите "Инициализировать чат"
  5. Кнопка чата должна появиться в правом нижнем углу
💡 Подсказка:

Чтобы получить токен, откройте в другом окне:

/frontend/chat/demo

На демо-странице будет показан токен и примеры использования.
