# Исходники для Mac App Store

Папка может быть **пустой**: тогда скрипт строит слоты Mac из **тех же** картинок, что iPhone/iPad (центральный кроп под широкий формат). Для витрины лучше положить сюда **отдельные** кадры окна приложения на macOS, если они есть.

Команда и выходные каталоги: [../README.md](../README.md).
