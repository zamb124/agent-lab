# Фаза 1 — инициация (магазины v1, матрица, учётки)

Заполняется командой; обновлять при смене стратегии дистрибуции.

## 1.1 Целевые магазины v1

| Магазин | Платформа | Формат | Приоритет v1 | Примечание |
|---------|-----------|--------|----------------|------------|
| Google Play | Android | TWA (AAB) | Да | Основной Android |
| Apple App Store | iOS / iPadOS | Capacitor (WKWebView) | Да | Обязательная оболочка |
| RuStore | Android | Тот же AAB/APK что Play | Опционально | Альтернативная витрина в РФ |
| Huawei AppGallery | Android | TWA/AAB | Опционально | По запросу рынка |
| Samsung Galaxy Store (Web Apps) | Android | Заявка по URL PWA | Опционально | Отдельный процесс |
| Microsoft Store | Windows | MSIX / PWA | Опционально | Отдельная дорожка |

**Решение v1 (зафиксировать):** минимум — Google Play + Apple App Store.

## 1.2 Матрица функций (оболочка vs веб)

| Функция | Android TWA (Chrome) | iOS Capacitor (WKWebView) | Примечание |
|---------|----------------------|---------------------------|------------|
| WebRTC / звонки (LiveKit) | Ожидается как в Chrome | Ожидается; проверить на устройстве | Info.plist: камера/микрофон |
| WebSocket / Sync | Да | Да | |
| In-app уведомления (WS → UI) | Да | Да | |
| Web Push (VAPID) | Как в Chrome | В Store-оболочке ограничено; см. PUSH_PARITY_APNS | |
| Загрузка файлов | Да | Да; ограничения iOS как в Safari | |
| Голосовые (MediaRecorder) | Да | Да; разрешение микрофона | |
| Фоновая геолокация | Ограничено PWA | Нет без нативного кода | Не целевой сценарий v1 |
