# Введение в Agent Lab

Agent Lab — это платформа для создания и управления ИИ-агентами на базе LangGraph. 
Платформа позволяет быстро создавать ботов, настраивать их поведение и подключать к различным каналам коммуникации.

## Основные возможности

### Создание агентов

- **Визуальный редактор** — создавайте агентов без написания кода
- **Поддержка различных LLM** — OpenAI, Anthropic, YandexGPT и другие
- **Гибкие промпты** — настраивайте поведение агента под ваши задачи

### Инструменты и интеграции

- **MCP серверы** — подключайте внешние инструменты через Model Context Protocol
- **Встроенные инструменты** — работа с файлами, HTTP запросы, базы знаний
- **RAG** — подключайте базы знаний для ответов на специфические вопросы

### Каналы коммуникации

- **Telegram** — полная интеграция с Telegram Bot API
- **WhatsApp** — подключение через WhatsApp Business API
- **Web-виджет** — встраиваемый чат для сайтов
- **REST API** — интеграция с любыми системами

### Управление

- **Переменные** — храните API ключи и настройки безопасно
- **История диалогов** — просматривайте и анализируйте разговоры
- **Биллинг** — отслеживайте использование и затраты

## Архитектура

Agent Lab построен на основе:

- **LangGraph** — фреймворк для создания агентов с состоянием
- **FastAPI** — высокопроизводительный веб-фреймворк
- **PostgreSQL** — надежное хранение данных
- **TaskIQ** — асинхронная обработка задач
