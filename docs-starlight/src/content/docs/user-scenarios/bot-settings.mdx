# Настройка бота

После установки бота вы можете настроить его параметры: описание, промпт, инструменты и каналы связи.

<Steps>

1. **Откройте раздел Боты**
   
   Откройте раздел **Боты** в боковом меню. Здесь отображается список всех ваших ботов.
   
   ![Открытие раздела Боты](/src/assets/bot-settings/01.png)

2. **Выберите бота**
   
   Нажмите на карточку бота, чтобы открыть его настройки.
   
   ![Выбор бота](/src/assets/bot-settings/02.png)

3. **Панель настроек**
   
   Откроется панель настроек бота с вкладками: Основные, Способности, MCP, База знаний, Каналы, Дополнительно.
   
   ![Панель настроек](/src/assets/bot-settings/03.png)

</Steps>

## Вкладки настроек

<Tabs>
  <TabItem label="Основные">
    На вкладке **Основные** можно изменить:
    - **Описание бота** — для понимания его назначения
    - **Промпт** — инструкции для ИИ
    - **LLM модель** — Claude, GPT-4, Gemini и другие
    
    ![Основные настройки](/src/assets/bot-settings/05.png)
  </TabItem>
  <TabItem label="Способности">
    На вкладке **Способности** управляйте инструментами бота:
    - Выбирайте доступные инструменты
    - Настраивайте параметры каждого инструмента
    
    ![Настройка инструментов](/src/assets/bot-settings/08.png)
  </TabItem>
  <TabItem label="Каналы">
    На вкладке **Каналы** подключите платформы:
    - **Telegram** — через Bot API
    - **WhatsApp** — через Business API
    - **Web** — встраиваемый виджет
    - **API** — REST интеграция
    
    ![Настройка каналов](/src/assets/bot-settings/10.png)
  </TabItem>
</Tabs>

## Пошаговая настройка

<Steps>

4. **Перейдите на вкладку Основные**
   
   ![Вкладка Основные](/src/assets/bot-settings/04.png)

5. **Отредактируйте описание**
   
   Введите **описание бота** — это поможет понять его назначение.
   
   ![Редактирование описания](/src/assets/bot-settings/06.png)

6. **Настройте способности**
   
   Перейдите на вкладку **Способности** и выберите инструменты для бота.
   
   ![Вкладка Способности](/src/assets/bot-settings/07.png)

7. **Настройте каналы**
   
   Перейдите на вкладку **Каналы** для подключения платформ.
   
   ![Вкладка Каналы](/src/assets/bot-settings/09.png)

8. **Выберите модель ИИ**
   
   В поле **LLM модель** выберите модель: Claude, GPT-4, Gemini и другие.
   
   ![Выбор модели ИИ](/src/assets/bot-settings/12.png)

9. **Сохраните настройки**
   
   Нажмите кнопку **сохранения** (иконка дискеты) для применения изменений.
   
   ![Сохранение настроек](/src/assets/bot-settings/13.png)

10. **Готово!**
    
    Настройки сохранены! Бот готов к работе с новыми параметрами.
    
    ![Настройки сохранены](/src/assets/bot-settings/14.png)

</Steps>

<Aside type="tip">
После изменения настроек бот сразу начнет использовать новые параметры для всех новых диалогов.
</Aside>
