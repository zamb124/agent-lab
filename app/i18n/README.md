# Директория переводов

Эта директория содержит все файлы системы интернационализации (i18n) для Agents Lab.

## 📁 Структура

```
app/i18n/
├── translations/               # Основные файлы переводов
│   ├── ru.json                # Русский (основной) - 100% заполнен
│   ├── en.json                # Английский - частично с [TODO: ключи]  
│   └── es.json                # Испанский - частично с [TODO: ключи]
│
├── keys/                      # Служебные файлы (автогенерированные)
│   ├── models.json            # Ключи из Pydantic моделей
│   ├── templates.json         # Ключи из HTML шаблонов
│   └── frontend.json          # Ключи из JavaScript файлов
│
├── generated/                 # JS модули для фронтенда (автогенерированные)
│   ├── ru.js                  # JavaScript модуль с переводами на русском
│   ├── en.js                  # JavaScript модуль с переводами на английском
│   └── es.js                  # JavaScript модуль с переводами на испанском
│
└── README.md                  # Этот файл
```

## 🔄 Автоматическая генерация

Все файлы в этой директории **автоматически генерируются** при запуске приложения:

1. **Сканирование кода** - система находит все ключи переводов в:
   - Pydantic моделях с `Field(title=...)`
   - HTML шаблонах с `{{ t('ключ') }}`
   - JavaScript файлах с `app.i18n.t('ключ')`

2. **Обновление JSON файлов** - новые ключи добавляются в `/translations/`

3. **Генерация JS модулей** - создаются оптимизированные модули в `/generated/`

## 📝 Редактирование переводов

### Ручное редактирование

Можно редактировать файлы в `translations/` напрямую:

```json
// Пример: app/i18n/translations/en.json
{
  "common": {
    "save": "Save",           // ✅ Переведено
    "cancel": "[TODO: common.cancel]"  // ❌ Требует перевода
  }
}
```

**Правила:**
- ✅ **Можно изменять** значения переводов 
- ❌ **Нельзя изменять** структуру ключей (они автогенерируются)
- ❌ **Нельзя удалять** секцию `meta`

### Маркировка переводов

- `[TODO: ключ]` - требует перевода
- Обычный текст - переведено
- `{параметр}` - подстановочные параметры

## 🚫 Что НЕ нужно редактировать

- **`/keys/`** - служебные файлы, перезаписываются при каждом запуске
- **`/generated/`** - JS модули, автогенерируются из JSON файлов
- **`meta` секции** - статистика файлов, вычисляется автоматически

## 🔧 Управление переводами

### Принудительная генерация

```bash
# Через API (нужны права админа)
curl -X POST http://localhost:8001/frontend/api/i18n/refresh

# При запуске приложения (автоматически)
uv run python run.py
```

### Статистика переводов

```bash
# Через API
curl http://localhost:8001/frontend/api/i18n/stats

# В консоли браузера
fetch('/frontend/api/i18n/stats').then(r => r.json()).then(console.log)
```

### Проверка полноты переводов

```bash
# Поиск непереведенных ключей
grep -r "\[TODO:" app/i18n/translations/

# Подсчет процента завершенности
jq '.meta.completeness' app/i18n/translations/en.json
```
