# Модуль Abilities (Способности)

## Описание

Модуль для отображения библиотеки доступных агентов и инструментов в системе Agent Lab.

## Возможности

- **Двухколоночный макет**: Агенты и инструменты отображаются в отдельных колонках
- **Единообразные карточки**: Все карточки имеют одинаковый размер (min-height: 220px)
- **Обрезка описаний**: Длинные описания обрезаются до 3 строк
- **Показ при наведении**: При наведении на карточку показывается полное описание
- **Фоновое изображение**: Hero секция с фоном из `/static/img/back.jpg`
- **Skeleton loaders**: Плавная загрузка с HTMX
- **Интернационализация**: Поддержка русского и английского языков
- **Адаптивность**: Отзывчивый дизайн для мобильных устройств

## Роуты

- `GET /frontend/abilities/` - Главная страница модуля
- `GET /frontend/abilities/list` - HTMX endpoint со списком агентов и тулов

## API Dependencies

Модуль использует следующие API endpoints:

- `/frontend/api/agents/?public_only=true` - Получение публичных агентов
- `/frontend/api/tools/?public_only=true` - Получение публичных инструментов

## Стили

### Основные CSS классы

- `.abilities-page` - Обёртка страницы
- `.abilities-hero` - Hero секция с фоном
- `.abilities-container` - Grid контейнер (2 колонки)
- `.abilities-column` - Колонка агентов/тулов
- `.abilities-grid` - Grid для карточек
- `.ability-card` - Карточка способности
- `.ability-description` - Описание с обрезкой текста

### CSS переменные

Использует CSS переменные из `variables.css`:

- `--bg-secondary` - Фон карточек
- `--border-color` - Цвет границ
- `--accent-primary` - Основной акцент
- `--text-primary` - Основной текст
- `--text-secondary` - Вторичный текст
- `--radius-lg` - Радиус скругления

## JavaScript API

### Глобальные методы

```javascript
// Показать детали способности в модальном окне с формой
await app.abilities.showDetails(id, type)
// id - ID агента или тула
// type - 'agent' или 'tool'

// Проверить что мы на странице Abilities
app.abilities.isAbilitiesPage()
```

### Модальное окно

При клике на карточку или кнопку "Подробнее" открывается модальное окно с формой объекта:

- **Агенты**: Загружается форма из `/frontend/models/agent/{id}?view=form`
- **Инструменты**: Загружается форма из `/frontend/models/tool/{id}?view=form`

Форма полностью интерактивна и позволяет:
- Просматривать все параметры агента/тула
- Редактировать поля (если есть права)
- Сохранять изменения через HTMX
- Использовать все существующие компоненты форм

## Использование

После перезапуска сервера модуль автоматически загружается через плагинную систему.

### В навигации

Модуль добавляет пункт в сайдбар:

- **ID**: `abilities`
- **Иконка**: `bi-star`
- **Порядок**: 30
- **Тип**: HTMX

### Кастомизация

Для изменения фонового изображения обновите путь в `abilities.css`:

```css
.abilities-hero {
    background-image: url('/static/img/your-image.jpg');
}
```

## Адаптивность

- **Desktop**: 2 колонки (агенты | тулы)
- **Tablet** (< 1024px): 1 колонка
- **Mobile** (< 768px): Уменьшенные отступы и размеры

## Особенности реализации

1. **Обрезка текста**: Использует `-webkit-line-clamp: 3` для ограничения описаний
2. **Hover эффект**: При наведении показывается полное описание через `overflow: visible`
3. **Единый размер карточек**: `min-height: 220px` для всех карточек
4. **Плавные анимации**: `transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1)`
